# Presets

To modify the presets, edit `src/presets.json`.

## File format

```
[
    {
        "name": "<PRESET NAME>",
        "segments": [
            {
                "color": "#<RRGGBB>",
                "height": <UINT>
            },
            {
                "color": "#<RRGGBB>",
                "height": <UINT>
            }
        ]
    },
    {
        "name": "<PRESET NAME>",
        "segments": [
            {
                "color": "#<RRGGBB>",
                "height": <UINT>
            },
            {
                "color": "#<RRGGBB>",
                "height": <UINT>
            }
        ]
    }
]
```
